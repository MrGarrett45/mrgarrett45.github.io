Track Trump

Project I made to track the percentage of the r/politics frontpage that was about Trump at any given time. Ran on Raspberry Pi. Uses the Reddit API to gather and post data, and matplotlib to create graphs. Results automatically published here.

This is an example of what the bot would post daily, recording the number of articles at each hour, daily highs/lows, as well as a daily graph and lifetime project graph which I will post below.
